Detailed Description

template<class T>
class DLList< T >

A custom doubly-linked list implementation.

A custom implementation of a doubly-linked list aimed at exploring the List data structure and its behavior, entirely for pedagogical purposes.

template<class T >
void DLList< T >::Delete ( )

Delete node.

Precondition
The list is non-empty and the cursor is not at the end of list.
Postcondition
The node pointed to by the cursor is deleted after the cursor points to next pointer. If cursor pointed to last item in list, then head and cursor are set to NULL.

Insert data after cursor.

Parameters
[in]itemData to be stored in list.
Postcondition
If list is empty, item is inserted at head of list. If cursor is at end of list, item is inserted as last item in the list. Otherwise, item is inserted at next pointer of cursor. Cursor then points to inserted item.
